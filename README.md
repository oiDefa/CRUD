# API de Músicas

## Descrição

API simples feita com Flask para cadastrar, listar, atualizar e excluir músicas usando um arquivo JSON.

---

## Como rodar

Instale o Flask:

```
pip install flask
```

Execute o projeto:

```
python app.py
```

Acesse:

```
http://127.0.0.1:5000
```

---


Os dados ficam no arquivo `musicas.json`.

*Não sei se funciona 100%*
